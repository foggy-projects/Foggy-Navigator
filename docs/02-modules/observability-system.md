## 1. жҰӮиҝ°

### 1.1 и®ҫи®Ўзӣ®ж Ү

еҸҜи§ӮеҜҹжҖ§зі»з»ҹпјҲObservability SystemпјүжҳҜ Foggy Navigator зі»з»ҹзҡ„йҮҚиҰҒз»„жҲҗйғЁеҲҶпјҢиҙҹиҙЈе®һж—¶зӣ‘жҺ§гҖҒиҝҪиёӘе’Ңз®ЎзҗҶзі»з»ҹзҡ„иҝҗиЎҢзҠ¶жҖҒгҖӮе…¶ж ёеҝғи®ҫи®Ўзӣ®ж ҮеҢ…жӢ¬пјҡ

- **е®һж—¶зӣ‘жҺ§**пјҡе®һж—¶зӣ‘жҺ§зі»з»ҹзҡ„иҝҗиЎҢзҠ¶жҖҒпјҢеҢ…жӢ¬дјҡиҜқгҖҒд»»еҠЎгҖҒе·Ҙе…·и°ғз”Ёзӯү
- **зҠ¶жҖҒиҝҪиёӘ**пјҡиҝҪиёӘжҜҸдёӘдјҡиҜқе’Ңд»»еҠЎзҡ„иҜҰз»ҶзҠ¶жҖҒе’Ңжү§иЎҢжөҒзЁӢ
- **д»»еҠЎжҺ§еҲ¶**пјҡж”ҜжҢҒд»»еҠЎзҡ„жҡӮеҒңгҖҒз»§з»ӯгҖҒдёӯжӯўгҖҒйҮҚиҜ•зӯүж“ҚдҪң
- **жҢҮж Үж”¶йӣҶ**пјҡж”¶йӣҶе’Ңеұ•зӨәзі»з»ҹзҡ„е…ій”®жҢҮж Ү
- **ж—Ҙеҝ—иҝҪиёӘ**пјҡжҸҗдҫӣе®Ңж•ҙзҡ„ж—Ҙеҝ—иҝҪиёӘе’Ңе®Ўи®ЎеҠҹиғҪ
- **е‘ҠиӯҰжңәеҲ¶**пјҡеҸҠж—¶еҸ‘зҺ°е’ҢйҖҡзҹҘзі»з»ҹејӮеёё
- **ж•°жҚ®еҸҜи§ҶеҢ–**пјҡжҸҗдҫӣзӣҙи§Ӯзҡ„ж•°жҚ®еҸҜи§ҶеҢ–з•Ңйқў
- **жҖ§иғҪеҲҶжһҗ**пјҡеҲҶжһҗзі»з»ҹжҖ§иғҪпјҢеҸ‘зҺ°з“¶йўҲ

#### 2.2.1 SessionMonitorпјҲдјҡиҜқзӣ‘жҺ§еҷЁпјү
