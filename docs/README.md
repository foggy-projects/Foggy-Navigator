## 📋 文档贡献

### 添加新文档
1. 确定文档所属的目录（01-overview、02-modules、03-implementation、04-research）
2. 在对应目录下创建新的 Markdown 文件
3. 更新本 README.md，添加文档链接
4. 遵循现有文档的格式和风格

### 文档命名规范
- 使用英文文件名，单词用连字符分隔（如：session-module.md）
- 文件名应简洁明了，反映文档内容
- 避免使用中文文件名

### 文档格式规范
- 使用 Markdown 格式
- 标题层级清晰（# 一级标题、## 二级标题、### 三级标题）
- 代码块使用语言标识（如 ```java、```sql）
- 表格用于展示对比和清单
- 使用适当的 Emoji 增强可读性
