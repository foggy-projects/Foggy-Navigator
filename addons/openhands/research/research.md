## 🔧 API 和集成

### CLI 工具

OpenHands 提供 CLI 工具，可以通过命令行使用：

```bash
# 安装
pip install openhands

# 使用
openhands "帮我实现一个用户登录功能"
```

### API 集成

OpenHands 提供 REST API，可以集成到现有系统中：

```python
from openhands import OpenHands

client = OpenHands(api_key="your-api-key")

result = client.execute_task(
    task="修改语义层 JavaScript 文件",
    context={
        "repository": "https://github.com/your-repo",
        "branch": "feature/update-semantic-layer",
        "files": ["semantic-layer.js"]
    }
)
```

### Docker 集成

```bash
# 运行 OpenHands 容器
docker run -d \
  -p 3000:3000 \
  -v /path/to/project:/workspace \
  allhandsai/openhands:latest
```
