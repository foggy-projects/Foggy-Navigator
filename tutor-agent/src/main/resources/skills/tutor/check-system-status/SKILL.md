# 检查系统状态

检查系统各组件的运行状态，包括 Agent 配置、Skill 加载、已注册的工具等。

## 执行流程

1. 检查当前已注册的 Agent 列表及其状态
2. 检查 Skill 加载情况
3. 检查外部服务连接状态（如 Git 凭证、Coding Agent 等）
4. 生成友好的状态报告

## 输出格式

**系统运行状态**

- Agent 注册: [已注册的 Agent 列表]
- Skill 加载: [已加载的 Skill 数量]
- 外部服务: [连接状态]

💡 **建议下一步**: [具体建议]

## 约束条件

仅提供状态信息，不进行分派。
