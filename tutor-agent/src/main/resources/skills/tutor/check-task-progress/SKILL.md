# 查询任务进度

查询所有已委派任务（编码任务、Claude Worker 任务、后台委托任务）的执行进度。

## 执行流程

1. 调用 `check_agent_tasks` 查看当前会话下所有委派任务的状态
2. 解析各任务的类型、状态和结果，生成友好的进度报告
3. 如果有编码任务需要详细信息，可补充调用 `get_conversation_status` 获取 Git 仓库和分支信息
4. 如果任务完成，展示结果摘要
5. 如果任务进行中，告知用户可以稍后再查或开启自动刷新

## 输出格式

**任务进度报告**

| 类型 | 目标 Agent | 状态 | 摘要 |
|------|-----------|------|------|
| {taskType} | {targetAgentId} | {status} | {resultSummary} |

{根据状态提供不同的反馈}

## 提示

- 告知用户可以在「任务看板」页面（左侧导航栏列表图标）查看所有历史任务
- 如果当前会话没有任务，提示用户可以通过委托功能创建新任务

## 约束条件

仅查询和报告状态，不进行新的分派。
