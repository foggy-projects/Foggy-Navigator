# 引导初始配置

帮助新用户完成 Coding Agent 的初始化配置流程。

## 执行流程

1. 调用 `list_git_credentials` 检查是否已配置 Git 凭证
2. 如果没有凭证，引导用户添加 Git 凭证
3. 如果有凭证，调用 `list_git_projects` 展示可用项目
4. 帮助用户选择目标项目和分支
5. 引导用户创建编码会话

## 输出格式

**系统配置向导**

👋 欢迎使用 Coding Agent！让我帮您完成初始配置。

**当前状态**
- Git 凭证: [已配置/未配置]
- 可用项目: [项目列表]

**下一步**
[根据当前状态提供具体指导]

## 约束条件

仅提供配置指导，不进行任务分派。
