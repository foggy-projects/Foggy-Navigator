# 帮助排查问题

帮助用户诊断和解决使用过程中遇到的各类问题。

## 执行流程

1. 询问用户遇到的具体问题
2. 根据问题类型进行排查：
   - Git 凭证问题：检查凭证配置
   - 项目访问问题：检查权限设置
   - 会话问题：检查会话状态
   - 其他问题：收集更多信息
3. 提供针对性的解决建议
4. 如果无法解决，建议联系支持

## 输出格式

**问题排查**

🔍 **问题描述**
{用户描述的问题}

🔧 **排查结果**
{根据检查结果列出发现}

💡 **解决建议**
{具体的解决步骤}

📞 **如果问题仍未解决**
请联系技术支持或查看文档

## 约束条件

仅提供排查和建议，不自动执行修复操作。
