# Skill ID
help-troubleshoot

# Skill标题
帮助排查问题

# 描述
帮助用户排查和解决使用过程中遇到的问题

# 触发条件
- 遇到问题
- 报错了
- 出错
- 帮助
- 怎么办
- 不工作
- 失败了

# 意图
- troubleshoot
- help
- error
- fix_problem

# 执行逻辑
1. 首先询问用户遇到的具体问题
2. 根据问题类型进行排查：
   - Git 凭证问题：检查凭证配置
   - 项目访问问题：检查权限设置
   - 会话问题：检查会话状态
   - 其他问题：收集更多信息
3. 提供针对性的解决建议
4. 如果无法解决，建议联系支持

# 输出格式
**问题排查**

🔍 **问题描述**
{用户描述的问题}

🔧 **排查结果**
{根据检查结果列出发现}

💡 **解决建议**
{具体的解决步骤}

📞 **如果问题仍未解决**
请联系技术支持或查看文档

# 分派条件
仅提供排查和建议，不自动执行修复操作
